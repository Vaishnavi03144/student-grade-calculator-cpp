# Student Grade Calculator (C++)

## 📌 Description
This is a simple C++ program that calculates a student's total marks, average, and grade based on user input.

## 🚀 Features
- Takes student name as input
- Accepts marks for multiple subjects
- Calculates total and average
- Assigns grade automatically

## 🧠 Concepts Used
- Input/Output (cin, cout)
- Loops (for loop)
- Conditional statements (if-else)
- Basic arithmetic operations

## ▶️ How to Run
1. Compile the code:
   g++ main.cpp -o grade

2. Run the program:
   ./grade

## 📷 Sample Output
Enter student name: Vaishnavi  
Enter number of subjects: 3  
Enter marks for subject 1: 90  
Enter marks for subject 2: 85  
Enter marks for subject 3: 88  

--- Result ---
Name: Vaishnavi  
Total Marks: 263  
Average: 87.67  
Grade: B
